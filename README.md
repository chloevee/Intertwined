# Intertwined

Intertwined is a capstone project developed as an informational website focused on modern dating apps and online relationships among Gen Z users. The website aims to bridge the gap between younger and older generations by providing accessible, educational, and engaging content for both Gen Z individuals and their parents.

## About the Project

As dating culture increasingly shifts into digital spaces, many parents struggle to understand how dating apps influence relationships, communication, and social behavior among younger generations. Intertwined was created to encourage awareness, understanding, and open conversations surrounding online dating culture.

The website presents information in a visually engaging and interactive format, making complex or sensitive topics easier to understand for a wider audience.

## Features

- Informational pages about dating apps and online relationships
- Interactive and responsive website design
- Modern UI tailored for Gen Z audiences
- Parent-friendly explanations and resources
- Multimedia assets including visuals and audio elements

## Technologies Used

- HTML
- CSS
- JavaScript

## Project Goal

The goal of Intertwined is to:
- Educate users about digital dating culture
- Promote safe and informed online interactions
- Encourage communication between Gen Z and parents
- Present information through an engaging digital experience
